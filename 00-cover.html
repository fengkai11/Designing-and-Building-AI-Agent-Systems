理论、架构、源码研究与工程实现

一本面向工程实践的 AI Agent 系统学习教程。

本书围绕 Agent 的核心机制展开：能力模型、Agent Loop、工具系统、上下文工程、Planning、Memory、RAG、长期任务、人机协同、评估、可观测性、源码研究、Runtime 设计与综合项目实现。

适合读者

  • 有编程基础，想系统进入 AI Agent 开发的人；
  • 用过 ChatGPT、Claude、Codex、Claude Code，但想理解背后机制的人；
  • 想从普通 LLM 应用开发升级到 Agent 系统设计的人；
  • 想构建个人助理、外贸获客 Agent、教育 Agent、代码 Agent、企业流程 Agent 的开发者。

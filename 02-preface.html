从零构建 AI Agent 系统

前言

这本书写给想真正构建 AI Agent 系统的人。

它不是一本提示词合集，也不是某个框架的 API 手册，而是一套围绕 Agent 系统能力建立的工程教程。全书从 Agent 的边界判断开始，逐步进入 Agent Loop、工具系统、上下文工程、Planning、Memory、RAG、长期任务、审批、评估、可观测性、源码研究和综合项目。

本书坚持一个基本立场：Agent 不是“会调用工具的聊天机器人”，也不是“无限自主的数字员工”。Agent 是由模型、工具、状态、上下文、记忆、权限、评估和人机协同共同组成的软件系统。

读者在阅读时，建议始终带着三个问题：

  1. 这个机制解决什么真实问题？
  2. 如果没有这个机制，Agent 会如何失败？
  3. 这个机制如何落实到代码、数据结构和产品界面？

如果你只是想快速体验一个 Demo，可以只读第 4、5、17 章。如果你想做真实产品，建议完整读完第 1–20 章，并完成练习汇总中的 12 个必做练习。

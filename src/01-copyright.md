# 版权与使用说明

本书为个人学习与项目实践整理版本，适合通过 mdBook 构建为在线电子书，并托管到 GitHub Pages。

## 使用建议

你可以将本仓库用于：

- 自己系统学习 AI Agent；
- 作为后续开发 Mini Agent Runtime 的需求文档；
- 作为 Codex / Claude Code 等代码智能体的工程实施参考；
- 在 GitHub 上持续迭代为公开电子书。

## 关于源码与事实边界

本书涉及 OpenClaw、Hermes 类助理、Claude Code、Codex 等内容时，应区分：

1. **公开事实**：来自项目公开文档、README、官方说明、可验证代码；
2. **合理推断**：基于公开行为、源码结构或产品形态做出的分析；
3. **架构启发**：本书从成熟项目中抽象出的设计模式，不等同于原项目实现。

## 图片说明

本书插图统一存放在 `src/figures/` 目录，并通过相对路径嵌入章节。使用 mdBook 构建时会自动复制到输出目录。
